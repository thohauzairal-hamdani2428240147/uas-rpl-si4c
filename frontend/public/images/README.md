# Folder Gambar JSC Arena Booking

Folder ini digunakan untuk menyimpan berkas gambar statis yang digunakan di seluruh halaman aplikasi JSC Arena Booking (misalnya: gambar lapangan futsal, basket, badminton, banner hero, dll.).

## Cara Menggunakan Gambar di React

Semua berkas gambar yang diletakkan di dalam folder `public/images` dapat diakses secara langsung di dalam kode komponen React menggunakan path absolut dari root web:

```jsx
// Contoh penggunaan gambar futsal.jpg di dalam komponen React:
<img src="/images/futsal.jpg" alt="Lapangan Futsal" />
```

## Daftar Rekomendasi Gambar
- `hero-banner.jpg` (Banner utama di halaman pemesanan)
- `futsal.jpg` (Pratampil Lapangan Futsal)
- `basketball.jpg` (Pratampil Lapangan Basket)
- `badminton.jpg` (Pratampil Lapangan Bulu Tangkis)
